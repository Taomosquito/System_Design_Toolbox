## Core Principle
Negating Goodharts Law With Metric Design

## Definition
This works by treating documentation as a source to compare actual actions against.
* In the event that the recipient doesn't do what they document the accuracy will suffer. 
* In the event that the recipient doesn't document what they do the accuracy will suffer. 
* In the event that the recipient miscategorizes the issue the accuracy will suffer. 

Outside of the overview there will be sample forms that put this front and center.
